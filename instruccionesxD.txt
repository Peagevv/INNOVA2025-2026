Hola, buenas noch@s

Debes instalar node.js: https://nodejs.org/es/download

Si todo sale bien verificalo con el comando:
    node -v (puede ser en el cmd o la terminal de VSC)

Despues verifica si tienes yarn con el comando:
    yarn -v (se supone que se instala cuando ya instalaste el node.js)
  sirve para instalar librerias, es de meta y la alternativa a el es npm.
para instalarlo usas el siguiente comando:
    npm install -g yarn //asi se instala de manera global, osea que este disponible en cualquier carpeta, es mas facil asi porque no tienes que 
    instalarlo en cada carpeta que vayas a usar

para instalar npm ocupamos una version sugerida: Node.js v20.x LTS
verificas la instalacion con los siguiente comandos:
    node -v
    npm -v
si ya tienes el Node lo instalas con el siguiente comando:
    npm install -g npm

RECOMENDACIŠON: VERIFICA LAS VARIABLES DE ENTORNO (PATH, asi te evitas contratiempos xD)
